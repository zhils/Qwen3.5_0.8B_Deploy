# 文本主干：24层调度与参数映射

## 1. 全局配置

- `hidden_size = 1024`
- `intermediate_size = 3584`
- `num_hidden_layers = 24`
- `rms_norm_eps = 1e-6`
- `hidden_act = silu`
- `tie_word_embeddings = true`

## 2. 每层固定骨架（Pre-Norm）

对第 `i` 层：

1. `h1 = RMSNorm(h, input_layernorm.weight)`
2. `attn_out = Attn_i(h1)`（按层类型选 linear/full）
3. `h = h + attn_out`
4. `h2 = RMSNorm(h, post_attention_layernorm.weight)`
5. `mlp_out = MLP(h2)`
6. `h = h + mlp_out`

## 3. 每层通用参数键

- `model.language_model.layers.{i}.mlp.up_proj.weight`
- `model.language_model.layers.{i}.mlp.down_proj.weight`

## 6. 层调度伪代码（含参数分支）

```python
def block_forward(i, h, w):
    h1 = rms_norm(h, w[f"layers.{i}.input_layernorm.weight"])
    if is_linear_layer(i):
        a = linear_attn_forward(i, h1, w)
    else:
        a = full_attn_forward(i, h1, w)
    h = h + a

    h2 = rms_norm(h, w[f"layers.{i}.post_attention_layernorm.weight"])
    m = mlp_forward(i, h2, w)
    h = h + m
    return h
```
